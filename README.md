## 项目介绍

基于 `electron` + `vite` + `vue` + `nodejs` 编写的具有闲鱼账号收发消息的桌面APP（仅在win端开发与测试）。

![预览图](docs/images/cover.png)

## 基本功能

- ✅多账号
- ✅消息提醒
- ✅自动回复
- ✅自定义消息处理逻辑
- ✅全局快捷键（Ctrl+Alt+Q 显示/隐藏窗口）
- ✅系统托盘支持

## 功能详情

### 🌟 系统托盘
- 应用关闭时最小化到系统托盘，不会完全退出
- 新消息到达时托盘图标闪烁提醒
- 右键托盘图标可快速访问主要功能

### ⌨️ 全局快捷键
- **Ctrl+Alt+Q** (Windows/Linux) 或 **Cmd+Alt+Q** (macOS)
- 在任何界面下快速显示/隐藏应用窗口
- 无需鼠标操作，提高使用效率

### 📱 消息管理
- 支持多个闲鱼账号同时在线
- 实时接收和发送消息
- 支持自动回复功能
- 可自定义消息处理逻辑

## 快速开始

### 安装依赖

```bash
yarn install
```

### 运行项目

```bash
yarn dev
```

### 打包 `.exe` 

```bash
yarn build:win
```

## 使用指南

### 🎮 快捷键操作
- **Ctrl+Alt+Q**：在任何时候快速显示或隐藏应用窗口
- 当应用隐藏到托盘时，使用此快捷键可快速唤起
- 当应用在前台时，使用此快捷键可快速隐藏到托盘

### 🔧 消息处理（自定义消息处理逻辑）

源代码在 `src/main/service/msg.service.ts` 中，请按需自行扩展。

### 📋 系统托盘操作
- 单击托盘图标：显示主窗口
- 右键托盘图标：打开功能菜单
- 托盘图标闪烁：有新消息提醒
